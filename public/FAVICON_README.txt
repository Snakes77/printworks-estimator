FAVICON SETUP INSTRUCTIONS
==========================

To add a proper favicon for DMC Encore:

1. Visit https://favicon.io/favicon-generator/
2. Upload the DMC Encore logo or create a simple icon
3. Download the favicon package
4. Copy favicon.ico to this directory (public/favicon.ico)
5. Optionally add other sizes (favicon-16x16.png, favicon-32x32.png)

For now, the app will work without a favicon (browser will just show a default icon).
